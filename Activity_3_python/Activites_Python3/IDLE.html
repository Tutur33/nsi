IDLE

IDLE est un environnement de développement intégré (Integrated Development Environment) pour Python. Un IDLE propose un certain nombre d'outils :


  • un éditeur de texte (pour écrire le programme)
  • un interpréteur (pour exécuter le programme)
  • un débogueur (pour tester le programme)
  • une fenètre intéractive REPL (Read-Evaluate-Print-Loop) permettant d'entrer du code Python arbitraire et de voir immédiatement les résultats.


Il existe des IDLE en ligne pour Python:

  • https://repl.it/languages/python3
  • https://trinket.io/
  • https://codewith.mu/
  • http://pythontutor.com/live.html#mode=edit permet de visualiser pas à pas l'exécution d'un programme Python
  • etc...


Il existe des IDLE installables sur les ordinateurs:

  • https://www.python.org/downloads/ site officiel
  • https://edupython.tuxfamily.org/ interface PyScripter traduite en français
  • https://thonny.org/ léger et efficace
  • https://codewith.mu/en/download
  • etc...
